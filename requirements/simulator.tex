\section{Simulator}
\label{sec:simulator}

\team{Immo Grabe}


(Interactive) simulation of a programs is its step-wise execution such that
the user can initate steps, choose among different alternatives \ldots and
can follow the execution on the editor. The simulator realizes the
\emph{semantics} from Section~\ref{sec:semantics}. The following points
should be implemented

\begin{itemize}
\item computation of a successor state
\item display of the taken step in the editor, using the highlight-function
  of the editor.
\end{itemize}


For an extended functionality, one could think of 
\begin{itemize}
\item interactive simulation, non-deterministic simulation
\item back-stepping
\item recording, storing, and replay of the taken steps
\end{itemize}
